## Project Rules

- Use `pnpm` for package-manager commands.
- Do not start a dev server unless the user explicitly asks.
- Prefer package-local scripts over broad workspace commands.
- Keep generated stack choices in `bts.jsonc` in sync with scaffold changes.
- For new Better Fullstack options, prefer the local Better Fullstack CLI or MCP
  update flow instead of hand-authoring unrelated starter code.
